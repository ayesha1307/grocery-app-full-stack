# 🛒 Grocery Budget Tracker

A full-stack web application to manage grocery expenses with category-wise budget tracking.

---

## 🚀 Features

- 💰 Set budget for categories (Fruits, Vegetables, Dairy, Snacks)
- 🛒 Add items to cart
- 🚨 Budget exceeded alert
- 📊 Budget progress tracker
- ❌ Delete items
- 🔄 Data stored in backend (Spring Boot API)

---

## 🧱 Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Spring Boot
- REST API

### Database
- MySQL (optional / future integration)
